This HTML file provides a portfolio of the graphs I made during my datascience masterclass with the Economics Observatory. I learnt how to use vega-lite to edit graphs, github to keep these as json files, and visual studio to code HTML and CSS. Everything I've made on visual studio needs to be saved into a local folder on my laptop, and all the graphs are already on my github ready to be pulled from. From there I can upload my 'index' to github and this updates my website!
